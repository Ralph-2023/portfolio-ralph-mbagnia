Dashboard Réduction du Gaspillage Alimentaire – Save N Serve

🎯 Contexte

Avec deux amis, nous avons conçu une solution innovante pour lutter contre le gaspillage alimentaire. L’objectif : créer un dashboard intelligent utilisant l’IA et le Machine Learning pour aider les restaurateurs à gérer leurs stocks et réduire leurs déchets.

🚀 Bénéfices

  • Réduction de 20 % des coûts d’achat en cuisine.
  • Optimisation du processus de gestion des stocks alimentaires.
  • Conformité avec les lois anti-gaspillage (AGEC, Garot).
  • Valorisation de l’engagement écoresponsable des restaurateurs.

🧰 Technologies utilisées

Frontend : React · Chart.js · Tailwind CSS
Backend : Node.js / Python
Base de données : MySQL
